# Node.js boilerplate

Getting Started
---------------
#### Useful commands

```
# Install NPM dependencies
npm install

# Start the app in the dev mode
npm run dev

#Linting
npm run lint
```

Prerequisites
-------------

- [MongoDB](https://www.mongodb.com/download-center/community)
- [Node.js 8.0+](http://nodejs.org)
